aconvasp_caller Module¶

Interface with command line aconvasp http://aflowlib.org/ Only tested on Linux inspired by Shyue’s qhull_caller WARNING: you need to have a convasp in your path for this to work

qhull_caller Module¶

Interface with command line qhull. Needs qhull installed. You can get it from http://www.qhull.org/.

As far as we know, no formal Python extension for higher dim hulls exist.

pymatgen.command_line.qhull_caller.qconvex(data)[source]¶

Input data should be in the form of a list of a list of floats. Returns the facets of the convex hull as a list of a list of integers.

pymatgen.command_line.qhull_caller.qvertex(data)[source]¶

Input data should be in the form of a list of a list of floats. Returns the facets of voronoi construction as a list of a list of float.

pymatgen.command_line.qhull_caller.qvertex_target(data, index)¶

Input data should be in the form of a list of a list of floats. index is the index of the targeted point Returns the vertices of the voronoi construction around this target point.

pymatgen.command_line.qhull_caller.qvoronoi(data)[source]¶

Input data should be in the form of a list of a list of floats. Returns voronoi results as a list of a list of integers.
