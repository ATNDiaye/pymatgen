Extended version of TransformedStructure to generates new materials from cifs.

Args:

Input cif string. Should contain only one structure. For cifs containing multiple structures, please use CifTransmuter.

Sequence of transformations to be applied to the input structure.

Option to set if the primitive cell should be extracted. Defaults to True. However, there are certain instances where you might want to use a non-primitive cell, e.g., if you are trying to generate all possible orderings of partial removals or order a disordered structure.

transmuters Module¶

This module implements various transmuter classes. Transmuters are essentially classes that generate TransformedStructures from various data sources. They enable the high-throughput generation of new structures and input files.

It also includes the helper function, batch_write_vasp_input to generate an entire directory of vasp input files for running.
